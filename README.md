# ge-mu — CONTOUR 草原と海

等高線でえがかれた地形（草原・砂浜・海）の内側を歩く、一人称／三人称の探索デモ（Three.js）。
`index.html` を開くだけで動きます（ビルド不要）。

明るい「草原と海」の世界に、マルチプレイ・アバター・設定などを融合したバージョンです。

## マルチプレイ（一緒に歩く）

他のプレイヤーもログインして、同じ地形を一緒に歩けます。**専用サーバーは不要**です。

### 遊び方
1. `index.html`（GitHub Pages などの HTTPS 上）を開く。
2. ゲート画面で **NAME**（表示名）と **ROOM**（合言葉）を入力し「あるきはじめる」。
3. 同じ **ROOM** 名を入力した仲間が、同じ世界に現れます。上部中央に接続状態と参加者一覧が出ます。

- 地形・岩の配置は全員で共有されます（同じ生成式・同じシード）。岩には当たり判定があります。
- 各プレイヤーは**サル（マカク）のアバター**として名前プレート付きで表示され、移動中は歩行アニメーションが付き、位置はなめらかに補間されます。
- ROOM を空にすると `lobby`（全員共通）になります。仲間内だけで遊ぶには、推測されにくい ROOM 名を決めてください。

### しくみ
- 位置情報は公開 MQTT ブローカー（`wss://broker.emqx.io`）越しに pub/sub します。
  `contour2/<ROOM>/state/<id>` に自分の座標を送信し、同じトピックを購読します。
- サーバー実装が不要なので、静的ホスティング（GitHub Pages 等）のまま動きます。
- 通信できない環境ではそのまま**一人プレイ**として動作します（`SOLO` 表示）。

> 注意: 公開ブローカーを使うため通信内容は暗号化されますが機密ではありません。
> 位置と表示名のみを送っており、個人情報は送信しません。

## 操作

**PC**: `WASD` 移動 · `SHIFT` 走る · `SPACE` 跳ぶ · クリックで視点ロック → マウスで視点 · `ESC` 解除

**スマホ / タブレット**: 画面左の**バーチャルスティック**で移動（大きく倒すと走る）·
画面ドラッグ（または右下の**視点パッド**）で視点 · **JUMP** ボタンで跳ぶ。
横向き（ランドスケープ）・ノッチにも対応します。

- 下中央の **3人称／1人称** ボタン（または操作モード **SP操作／PC操作**）を切り替えられます。
- 三人称ではサルのアバターを後方から見られます（仲間のアバターも一緒に見えます）。
- 海（標高 0 m の等高線）に潜ると画面が青くにじみます。

## バトル（撃ち合い）

プレイヤーは銃を持ち、同じ ROOM の相手を撃ち合えます。

- **射撃**: PC はマウス左クリック（押しっぱなしで連射）、スマホは **FIRE** ボタン。画面中央のクロスヘアで狙います。地形や岩に隠れると弾は当たりません。
- **HP**: 各プレイヤーは HP 100。被弾すると減り、頭上の HP バー（他プレイヤー）／左上の HP バー（自分）に反映されます。1 発 20 ダメージ。
- **ゲームオーバー**: HP が 0 になると GAME OVER。**コンティニュー**ボタン（PC は Enter キーでも可）で HP 全回復＆別地点に復活します。
- **透明化（ステルス）**: **透明** ボタン（PC は `Q` キー）で 5 秒間、他プレイヤーから姿と名前が見えなくなります。ただし無敵ではなく、被弾すればダメージは受けます。透明中も攻撃は可能で、自分の画面ではアバターが半透明で見えます。使用後 30 秒はクールダウン。

> 通信の仕組み: 撃った側が命中を判定して命中メッセージを送り、撃たれた側が自分の HP を減らして全員に共有します（サーバー不要のまま成立）。

## 設定

右上の **⚙ ボタン**で設定を開けます。
- **画面の明るさ / BRIGHTNESS**: スライダーで画面全体の明るさを調整できます（初期値 100%）。設定はブラウザに保存され、次回も引き継がれます。

> PC ではプレイ中マウスがゲームに固定されるため、`ESC` で一度カーソルを解放してから ⚙ を押してください。スマホでは常にタップできます。
