# ge-mu — CONTOUR 市街戦

ビルの立ちならぶ**都市**を舞台にした、一人称／三人称のオンライン対戦シューター（Three.js）。
`index.html` を開くだけで動きます（ビルド不要）。

同じ ROOM の仲間と撃ち合い、キル数をきそいます。最多キルのプレイヤーには王冠がつきます。

## マルチプレイ（一緒に歩く）

他のプレイヤーもログインして、同じ地形を一緒に歩けます。**専用サーバーは不要**です。

### 遊び方
1. `index.html`（GitHub Pages などの HTTPS 上）を開く。
2. ゲート画面で **NAME**（表示名）と **ROOM**（合言葉）を入力し「あるきはじめる」。
3. 同じ **ROOM** 名を入力した仲間が、同じ世界に現れます。上部中央に接続状態と参加者一覧が出ます。

- 街（ビルの配置）は全員で共有されます（同じ生成式・同じシード）。ビルには当たり判定があり、射線もさえぎります。
- 舞台は**渋谷風の街並み**（Shibuya スタイル）: 中央に**スクランブル交差点**、車道・歩道・**横断歩道**・車線、**並木・街灯・信号機**を配置しています。※実在の地図データに基づく厳密な再現ではなく、雰囲気を再現した街です。
- 各プレイヤーは**サル（マカク）のアバター**として名前プレート付きで表示され、移動中は歩行アニメーションが付き、位置はなめらかに補間されます。
- ROOM を空にすると `lobby`（全員共通）になります。仲間内だけで遊ぶには、推測されにくい ROOM 名を決めてください。

### しくみ
- 位置情報は公開 MQTT ブローカー（`wss://broker.emqx.io`）越しに pub/sub します。
  `contour2/<ROOM>/state/<id>` に自分の座標を送信し、同じトピックを購読します。
- サーバー実装が不要なので、静的ホスティング（GitHub Pages 等）のまま動きます。
- 通信できない環境ではそのまま**一人プレイ**として動作します（`SOLO` 表示）。

> 注意: 公開ブローカーを使うため通信内容は暗号化されますが機密ではありません。
> 位置と表示名のみを送っており、個人情報は送信しません。

## 操作

**PC**: `WASD` 移動 · `SHIFT` 走る · `SPACE` 跳ぶ · クリックで視点ロック → マウスで視点 · `ESC` 解除

**スマホ / タブレット**: 画面左の**バーチャルスティック**で移動（大きく倒すと走る）·
画面ドラッグ（または右下の**視点パッド**）で視点 · **JUMP** ボタンで跳ぶ。
横向き（ランドスケープ）・ノッチにも対応します。

- 下中央の **3人称／1人称** ボタン（または操作モード **SP操作／PC操作**）を切り替えられます。
- 三人称ではサルのアバターを後方から見られます（仲間のアバターも一緒に見えます）。
- 海（標高 0 m の等高線）に潜ると画面が青くにじみます。

## バトル（撃ち合い）

プレイヤーは銃を持ち、同じ ROOM の相手を撃ち合えます。

- **射撃**: PC はマウス左クリック（押しっぱなしで連射）、スマホは **FIRE** ボタン。画面中央のクロスヘアで狙います。地形や岩に隠れると弾は当たりません。
- **HP**: 各プレイヤーは HP 100。被弾すると減り、頭上の HP バー（他プレイヤー）／左上の HP バー（自分）に反映されます。1 発 20 ダメージ。頭上の HP バーは**建物などに遮られると見えなくなります**（壁越しには見えません）。
- **ゲームオーバー**: HP が 0 になると GAME OVER。**コンティニュー**ボタン（PC は Enter キーでも可）で HP 全回復＆別地点に復活します。
- **透明化（ステルス）**: **透明** ボタン（PC は `Q` キー）で 5 秒間、他プレイヤーから姿と名前が見えなくなります。ただし無敵ではなく、被弾すればダメージは受けます。透明中も攻撃は可能で、自分の画面ではアバターが半透明で見えます。使用後 30 秒はクールダウン。
- **キル数と王冠**: 相手を倒すとキル数が加算されます（**街の猿を倒してもキル数に入ります**）。そのルームで**最もキル数の多いプレイヤー**の名前の横（および頭上）に**👑 王冠**が付き、周囲からも見えます。
- **回復（バナナ）**: 歩道にときどき**バナナ**が落ちています。拾うと **HP が 40 回復**します（上限 100）。拾ったバナナはしばらくすると同じ場所に**復活**します。
- **効果音**: 発砲・ジャンプ・被弾・死亡・回復に効果音が付きます。**他プレイヤーの発砲音は距離に応じて音量が変化**（近いほど大きく、遠いほど小さく）し、左右の定位も付きます。
- **被弾エフェクト**: 被弾すると画面が赤くフラッシュし、カメラが揺れます。

> 通信の仕組み: 撃った側が命中を判定して命中メッセージを送り、撃たれた側が自分の HP を減らして全員に共有します。キルは、倒された側が「誰に倒されたか」を通知し、その相手のキル数が加算されます（サーバー不要のまま成立）。
>
> ラグ対策: 各プレイヤーは位置に加えて**速度**を配信し、受信側は最後の位置を速度で**先読み（デッドレコニング）**して補間します。これにより相手が「少し過去の位置」ではなく現在位置に近い所に表示され、近距離の撃ち合いが成立しやすくなります。送信間隔も約18Hzに上げています。ただし公開ブローカー経由のため往復遅延自体は残ります（根本解決には専用の低遅延中継＝WebSocket/WebRTC が必要）。

## 街と交通（モブ）

街には信号で制御される交通が流れています（見た目の賑わい用。プレイヤーとは当たりません）。

- **信号機**: 車両用（横型3灯：青・黄・赤、赤が右）と歩行者用（2灯）を再現。**交差点の四隅すべてに設置**（1交差点=4基）。**交差点ごとに位相をずらして**あり（グリーンウェーブ）、各交差点のレンズ色が個別に変化します。
- **車（約140台）**: **作り込んだ車種**（セダン・軽・バン・タクシー）で、タイヤ・窓ガラス・ヘッド／テールライトを備えます。車線を走り、赤信号や前方の車で**停止・発進**、交差点で直進・左右折します。**車にはねられるとダメージ**（速い車ほど痛い）。マップ全体に分散しています。
- **歩行者（約220人）**: **二関節で歩く人間**（ひざ・ひじが曲がり、胴・頭・髪・足つき）。歩く人／走る人がいて、横断歩道では信号に従って渡ります。マップ全体に分散。
- **フェラーリ（高速スポーツカー）**: 赤い高速車が**6台、街全体を走り回ります**（各エリアから出発）。**他の車の約3倍**の速度で、**交差点を曲がるときは必ずドリフト**（スリップ＋スキッドマーク）します。前に遅い車がいるときは**車間を空けて追従**し、他の車にめり込みません。
- **猿（我々と同じアバター）**: **十数体**が歩いています。**誤って撃つと逆上して追いかけてきて殴られます**（ダメージ）。
- **並木・街灯・ネオン看板**: 歩道沿いに並木と街灯、ビルには点滅するネオン看板が並びます。

> 群衆はインスタンシングで描画し、大量でも軽量に動かしています。

> 交通は各自の画面でローカルに動く演出です（プレイヤー同士の位置・戦闘は同期します）。

## 音楽（BGM）

BGM が流れます。すべて**その場で合成**しており（音源ファイル不要・CSP セーフ）、音量は設定の **BGM VOLUME** で**効果音とは別に**調整できます（**SOUND** オフで BGM も止まります）。

- **ふだん**: **ほのぼのとした穏やかな曲**（メジャーのパッド＋ベル）。
- **他プレイヤーが近づくと（約45m 以内）**: **激しいデスロック系**（歪ませたパワーコードの刻み＋ドライブするキック／スネア）に切り替わります。
- 2 つのレイヤーは**クロスフェード**するので、相手が近づく／離れるのに合わせて滑らかに移り変わります。

## マップ

- **ミニマップ**: 画面**左上**に街のミニマップが出ます。道路のグリッド、**自分（緑の矢印＝向き付き）**、**他プレイヤー（赤）**、**バナナ（黄）**が表示されます（北が上・東が右）。
- **全体マップ**: ミニマップを**タップ**すると全画面の大きなマップが開きます。もう一度**タップで閉じます**。

## チャット

同じ ROOM の仲間と文字で会話できます。

- 画面左の **💬 ボタン**でチャットを開き、メッセージを入力して**送信**（PC は Enter でも送信）。
- 受信したメッセージは**チャット欄**に流れ、画面左下に**最新メッセージが数秒間**表示されます。未読があると 💬 ボタンに赤い印が付きます。
- 通信は公開 MQTT の `chat` トピック越しです（サーバー不要）。オフライン時は自分の画面内だけで表示されます。

## 設定

右上の **⚙ ボタン**で設定を開けます。設定はブラウザに保存され、次回も引き継がれます。
- **画面の明るさ / BRIGHTNESS**: スライダーで画面全体の明るさを調整できます（初期値 100%）。
- **効果音 / SOUND**: 効果音と BGM のオン／オフを切り替えられます。
- **効果音の音量 / SFX VOLUME**: 効果音の音量を調整できます（初期値 70%）。
- **音楽の音量 / BGM VOLUME**: BGM の音量を**効果音とは別に**調整できます（初期値 50%）。

> PC ではプレイ中マウスがゲームに固定されるため、`ESC` で一度カーソルを解放してから ⚙ を押してください。スマホでは常にタップできます。
