# ge-mu — CONTOUR 測量野帳

等高線だけで描かれた地形の内側を歩く、一人称の探索デモ（Three.js）。
`index.html` を開くだけで動きます（ビルド不要）。

## マルチプレイ（一緒に歩く）

他のプレイヤーもログインして、同じ地形を一緒に歩けます。**専用サーバーは不要**です。

### 遊び方
1. `index.html`（GitHub Pages などの HTTPS 上）を開く。
2. ゲート画面で **NAME**（表示名）と **ROOM**（合言葉）を入力し「ゲームスタート」。
3. 同じ **ROOM** 名を入力した仲間が、同じ世界に現れます。左上に接続状態と参加者一覧が出ます。

- 地形は全員で共有されます（同じ生成式・同じシード）。
- 入室すると人型アバターが自動生成され、名前プレート付きで表示されます。移動中は歩行アニメーションが付き、位置はなめらかに補間されます。
- ROOM を空にすると `lobby`（全員共通）になります。仲間内だけで遊ぶには、推測されにくい ROOM 名を決めてください。

### しくみ
- 位置情報は公開 MQTT ブローカー（`wss://broker.emqx.io`）越しに pub/sub します。
  `contour/<ROOM>/state/<id>` に自分の座標を送信し、同じトピックを購読します。
- サーバー実装が不要なので、静的ホスティング（GitHub Pages 等）のまま動きます。
- 通信できない環境ではそのまま**一人プレイ**として動作します（左上は `SOLO` 表示）。

> 注意: 公開ブローカーを使うため通信内容は暗号化されますが機密ではありません。
> 位置と表示名のみを送っており、個人情報は送信しません。

## 操作

**PC**: `WASD` 移動 · `SHIFT` 走る · `SPACE` 跳ぶ · マウス 視点 · `V` 一人称／三人称 切替 · `ESC` 視点解除

**スマホ / タブレット**: 画面左の**バーチャルスティック**で移動（大きく倒すと走る）·
画面右を**ドラッグ**して視点 · **JUMP** ボタンで跳ぶ · **視点** ボタンで一人称／三人称を切り替え。
タッチ対応端末では自動でオンスクリーン操作に切り替わり、横向き（ランドスケープ）にも対応します。

三人称視点では自分のアバターを後方から見られます（マルチプレイでは仲間のアバターも一緒に見えます）。

## 設定

右上の **⚙ ボタン**で設定を開けます。
- **画面の明るさ / BRIGHTNESS**: スライダーで画面全体の明るさを調整できます（初期値 130%）。暗くて周りが見づらいときに上げてください。設定はブラウザに保存され、次回も引き継がれます。

> PC ではプレイ中マウスがゲームに固定されているため、`ESC` で一度カーソルを解放してから ⚙ を押してください。スマホでは常にタップできます。
